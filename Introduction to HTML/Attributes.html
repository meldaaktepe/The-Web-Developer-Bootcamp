Attributes

Attributes are additional values that configures the elements. Tjey use to change or adjust their behavior or providing metadata in various ways.
An attribute always has the form name="value" (the attribute's identifier followed by its associated value). (From MDN)

Images

For displaying images "img" tag is used. This tag is self closing tag, therefore we don't nead to close the tag. It has a attribute named src, this atribute is used for referencing source file or url of the image that we want to use

Anchor Tag

Anchor tag is used for creating hyperlinks to other websides, other files in the same page or in new page. It is an inline tag. To make a link it uses href attribute link's to destination that can be gone. Also clicleble tex has to be given to work.
