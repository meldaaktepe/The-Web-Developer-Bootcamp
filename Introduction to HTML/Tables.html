Table

To display the data in table format "table" tag is used. Like list elements table element has some special tags that can be used. Some tags like tr and td are must, thead, tbody and th are nice to added them. Just like tml it has a head and title, title is generally defined in head section of the table

thead

Defines the head of the colums of the table

tbody

Like Html's body it defines the rows of the table wich the actual data is shown

th

It add's a heading to each colum. it is preferd to defined in thead section

tr

Defines a row of cells in a table.

td

Defines a colum for a table that contains data. It participates in the table model.
